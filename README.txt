
VAULT WebApp Mockup
-------------------

Содержит 3 страницы (index.html, category.html, product.html) и стили js.
Дизайн: черный минимализм.

Интеграция (что нужно подключить в продакшне):
- SDEK: получить clientId/clientSecret, использовать метод расчета стоимости доставки по городу/весу. В app.js сейчас заглушка calcShipping().
- Tinkoff Kassa: подключить API, создать платёж и редирект на paymentUrl. Обработать callback (notify) для подтверждения оплаты.
- Чеки: для самозанятых — формирование и отправка чека через API Tinkoff (или сервис оператора фискализации).

Категории (примеры):
- Мужские, Женские, Кроссовки, Кофты, Штаны, Аксессуары, Новинки, Распродажа, Утеплённые, Летняя линия

Файлы готовы для загрузки в репозиторий GitHub Pages.
